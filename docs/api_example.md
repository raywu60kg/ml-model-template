# API example
## Health Check
```
curl -X GET "http://localhost:8000/health" -H  "accept: application/json"
```

## Get Model Metrics
```
curl -X GET "http://localhost:8000/model/metrics" -H  "accept: application/json"
```

## Retrain Model
```
curl -X PUT "http://localhost:8000/model" -H  "accept: application/json"
```
